# Shadow Monarch / Kurayami Launcher

An open-source custom League of Legends skin manager and UI launcher, complete with a legacy FFI-based DLL injection patcher.

## Structure
- `shadow-monarch`: The elegant Electron-based UI and Node.js backend.
- `cslol-patcher-legacy`: The open-source C implementation of the injection DLL used to load custom skins into the game.
